## Getting started

```
$ git clone https://github.com/TerriaJS/TerriaMap.git
$ cd TerriaMap
$ export NODE_OPTIONS=--max_old_space_size=4096
$ npm install -g yarn
$ yarn install && yarn gulp && yarn start
# Open at http://localhost:3001
```
